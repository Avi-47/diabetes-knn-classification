# Diabetes Prediction with KNN

This project implements a K-Nearest Neighbors (KNN) classifier 
to predict diabetes using the Pima Indians dataset.

## Dataset
Pima Indians Diabetes Dataset

## Steps
- Data Preprocessing
- Standard Scaling
- KNN Model
- Cross Validation
- Hyperparameter Optimization (GridSearchCV)

## Results
- CV Accuracy: ~0.78
- CV ROC AUC: ~0.83

## Technologies
- Python
- Pandas
- Scikit-learn
